class

tab-snip% : class?

  superclass: string-snip%

将选项卡插入编辑器时，将自动创建tab-snip%的实例。

constructor

(new tab-snip%) → (is-a?/c tab-snip%)

为单个选项卡创建一个剪切，尽管该选项卡最初是空的。

通常，使用insert方法将单个选项卡插入到tab-snip%对象中。

在编辑器中确定单个字符的大小时不绘制选项卡的内容，其中选项卡由字符宽度确定（请参见set-tabs）；如果内容是单个选项卡字符（正常情况下），则将剪切字体的平均字符宽度用作标签的宽度。
