make-object
ok?
    ← prev  up  next → 

class

cursor% : class?

  superclass: object%

光标是指示鼠标指针位置的小图标。位图图像通常指示鼠标单击当前位置的当前模式或含义。

将为每个窗口指定一个光标（或窗口可以使用其父窗口的光标；有关详细信息，请参见set-cursor），当指针移到窗口上时，指针图像将更改为与窗口的光标匹配。每个光标对象可以分配给多个窗口。

constructor

(make-object cursor% image    
  mask    
  [hot-spot-x    
  hot-spot-y]) → (is-a?/c cursor%)
  image : (is-a?/c bitmap%)
  mask : (is-a?/c bitmap%)
  hot-spot-x : (integer-in 0 15) = 0
  hot-spot-y : (integer-in 0 15) = 0
(make-object cursor% id) → (is-a?/c cursor%)
  id : 
(or/c 'arrow 'bullseye 'cross 'hand 'ibeam 'watch 'blank
      'size-n/s 'size-e/w 'size-ne/sw 'size-nw/se)

第一种情况是使用图像位图和遮罩位图创建光标。两个位图的深度必须为1，大小必须为16 x 16像素。hot-spot-x和hot-spot-y参数确定光标图像中光标相对于其左上角的焦点。

第二种情况是使用库存光标创建一个光标，指定为以下内容之一：

  • 'arrow——默认光标

  • 'bullseye——同心圆

  • 'cross——十字线

  • 'hand——张开的手

  • 'ibeam——一条竖线,指示单击可控制文本选择插入符号

  • 'watch——一个手表或沙漏,表示用户必须等待计算完成

  • 'arrow+watch——带有手表或沙漏的默认光标，指示正在进行某些计算，但光标仍可以使用

  • 'blank——不可见

  • 'size-e/w——左右箭头

  • 'size-n/s——上下箭头

  • 'size-ne/sw——箭头向上向右和向下向左

  • 'size-nw/se——左上和右下箭头

如果光标创建成功，ok?返回#t，否则不能将光标对象分配给窗口。

method

(send a-cursor ok?) → boolean?

如果光标可以分配给窗口,则返回#t,否则返回#f。
