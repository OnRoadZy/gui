interface

list-control<%> : interface?

  implements: control<%>

列表控件为用户提供要从中选择的字符串项列表。有两个内置类实现list-control<%>：

  • choice% ——在弹出菜单中显示列表（这样用户一次只能选择一个项目）。

  • list-box% ——在滚动框中显示列表，允许使用选择一个项目（如果样式包括'single）或任意数量的项目。

在这两种情况下，都可以动态更改用户可选择的项目集。

method

(send a-list-control append item) → void?

  item : label-string?
将新项目添加到用户可选项目列表中。当前选择保持不变（除非列表控件是空的选择控件，在这种情况下选择新项）。

method

(send a-list-control clear) → void?

从控件中删除所有用户可选择的项。

method

(send a-list-control delete n) → void?

  n : exact-nonnegative-integer?

删除按n索引的项（其中项是从0索引的）。如果n等于或大于控件中的项数，an exn:fail:contract exception is raised。

未删除的选定项目将保持选定状态，并且不选择其他项目。

method

(send a-list-control find-string s)

 → (or/c exact-nonnegative-integer? #f)
  s : string?
查找与给定字符串匹配的用户可选项。如果找不到匹配选项，则返回#f，否则返回匹配选项的索引（其中项从0索引）。

method

(send a-list-control get-number) → exact-nonnegative-integer?

返回控件中用户可选择的项的数目（这也是列表控件中最大索引的一个以上）。

method

(send a-list-control get-selection)

 → (or/c exact-nonnegative-integer? #f)
返回当前选定项的索引（其中项是从0索引的）。如果选项项当前不包含选项或没有选择，则返回#f。如果允许多个选择并且选择了多个项目，则返回第一个选择的索引。

method

(send a-list-control get-string n)

 → (and/c immutable? label-string?)
  n : exact-nonnegative-integer?

返回给定索引的项（其中项是从0索引的）。如果提供的索引大于列表控件中的最大索引，an exn:fail:contract exception is raised。

method

(send a-list-control get-string-selection)

 → (or/c (and/c immutable? label-string?) #f)
返回当前选定的项。如果控件当前不包含任何选项，则返回#f。如果允许多个选择并且选择了多个项目，则返回第一个选择。

method

(send a-list-control set-selection n) → void?

  n : exact-nonnegative-integer?
选择由给定索引指定的项（其中项是从0索引的）。如果给定的索引大于列表控件中的最大索引，an exn:fail:contract exception is raised。

在列表框控件中，即使控件中允许多个选择，也会取消选择所有其他项。另请参见select in list-box%。

调用此方法时，不会调用控件的回调过程。

列表控件选项可以被用户单击控件更改，这样的更改不会通过这个方法；使用control回调过程（作为初始化参数提供）来监视选项更改。

method

(send a-list-control set-string-selection s) → void?

  s : string?
选择与给定字符串匹配的项。如果在列表控件中找不到匹配项，an exn:fail:contract exception is raised。

在列表框控件中，即使控件中允许多个选择，也会取消选择所有其他项。另请参见select in list-box%。

调用此方法时，不会调用控件的回调过程。

列表控件选项可以被用户单击控件更改，这样的更改不会通过这个方法；使用control回调过程（作为初始化参数提供）来监视选项更改。
