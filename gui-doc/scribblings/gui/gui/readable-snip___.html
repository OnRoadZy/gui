interface

readable-snip<%> : interface?

readable-snip<%>对象由open-input-text-editor生成的端口进行特殊处理：当输入流遇到readable-snip<%>对象时，调用其read-special方法来生成剪切的读取结果，该结果从端口返回为read-char-or-special意义上的“特殊”值。

由于read和read-syntax基于read-char-or-special可以实现readable-snip<%>以便在包含剪切的流上使用read时生成完整的S表达式或其他类型的值。

method

(send a-readable-snip read-special source    
  line    
  column    
  position) → any/c
  source : any/c
  line : (or/c exact-nonnegative-integer? #f)
  column : (or/c exact-nonnegative-integer? #f)
  position : (or/c exact-nonnegative-integer? #f)

这些参数与自定义输入端口的read-in返回的过程的参数相同；有关详细信息，请参阅(part ("(lib scribblings/reference/reference.scrbl)" "customport"))。结果也与read-in生成的过程的结果相同。
