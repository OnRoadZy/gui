class

snip-admin% : class?

  superclass: object%

有关管理员角色的信息，请参阅管理员。snip-admin%类从未直接实例化。它甚至没有被大多数程序员通过派生类实例化；每个text%或pasteboard%对象创建自己的管理员。但是，派生此类的新实例以在新上下文中显示剪切可能很有用。此外，从拥有的剪切调用现有管理员的方法可能很有用。

要创建新的snip-admin%类，必须重写此处描述的所有方法。它们都是由管理员的剪切调用的。

由于snip-admin%对象通常拥有多个剪切，许多方法需要snip%对象作为参数。

constructor

(new snip-admin%) → (is-a?/c snip-admin%)

创建（无用的）编辑器管理员。

method

(send a-snip-admin get-dc) → (or/c (is-a?/c dc<%>) #f)

获取适用于确定显示大小信息的绘图上下文。如果没有显示剪切，则返回#f。

method

(send a-snip-admin get-editor)

 → (or/c (is-a?/c text%) (is-a?/c pasteboard%))

返回此管理员报告的编辑器（直接或间接）。

method

(send a-snip-admin get-view x y w h [snip]) → void?

  x : (or/c (box/c real?) #f)
  y : (or/c (box/c real?) #f)
  w : (or/c (box/c (and/c real? (not/c negative?))) #f)
  h : (or/c (box/c (and/c real? (not/c negative?))) #f)
  snip : (or/c (is-a?/c snip%) #f) = #f
Specification:

规格：获取剪切坐标中剪切可见区域的定位（location）和大小。如果给定的剪切不由该管理员管理，则结果未定义。

如果snip不是#f，则剪切的当前可见区域安装在框x、y、w和h中。x和y值相对于剪切的左上角。w和h值可能大于剪切本身。

如果snip为#f，则剪切顶级显示的总可见区域将以编辑器坐标返回。使用#f进行snip类似于在get-view in editor-admin%使用#t进行full?。

如果未指定剪切，则剪切编辑器的定位和大小将返回到编辑器坐标中。

另请参见get-view in editor-admin%。

Default implementation:

默认实现：用0.0填充所有框。

method

(send a-snip-admin get-view-size w h) → void?

  w : (or/c (box/c (and/c real? (not/c negative?))) #f)
  h : (or/c (box/c (and/c real? (not/c negative?))) #f)
Specification:

规格：获取管理员显示区域的可见大小。

如果显示为编辑器画布，也请参见reflow-container。

Default implementation:

默认实现：用0.0填充所有框。

method

(send a-snip-admin modified snip modified?) → void?

  snip : (is-a?/c snip%)
  modified? : any/c
Specification:

指定：由剪切调用以报告其修改状态已更改为已修改或未修改。

Default implementation:

默认实现：不执行任何操作。

method

(send a-snip-admin needs-update snip    
  localx    
  localy    
  w    
  h) → void?
  snip : (is-a?/c snip%)
  localx : real?
  localy : real?
  w : (and/c real? (not/c negative?))
  h : (and/c real? (not/c negative?))
Specification:

规范：由剪切调用以请求剪切的显示需要更新。管理员决定何时实际更新剪切；最后调用剪切的draw方法。

localx、localy、w和h参数指定要刷新的剪切区域（在剪切坐标中）。

如果给定的剪切不由该管理员管理，则不会发生更新。

Default implementation:

默认实现：不执行任何操作。

method

(send a-snip-admin popup-menu menu snip x y) → boolean?

  menu : (is-a?/c popup-menu%)
  snip : (is-a?/c snip%)
  x : real?
  y : real?
Specification:

规格：在显示上打开一个弹出菜单，用于此剪切编辑器。如果弹出菜单成功，则结果为#t，否则为#f（与用户是否在弹出菜单中选择项目无关）。

菜单位于snip坐标中的x和y处。

当弹出菜单时，它的目标被设置为该剪切编辑器显示的顶级编辑器。有关详细信息，请参见get-popup-target。 Default implementation:

默认实现：返回#f。

method

(send a-snip-admin recounted snip refresh?) → void?

  snip : (is-a?/c snip%)
  refresh? : any/c
Specification:

规范：由剪切调用，通知管理员指定的剪切已更改其计数。剪切通常需要在更改其计数后更新，但剪切决定是否立即更新。

如果refresh?不是#f，则剪切请求立即更新。否则，needs-update也必须调用需求更新。

如果给定的剪切不由该管理员管理，则忽略方法调用。

Default implementation:

默认实现：不执行任何操作。

method

(send a-snip-admin release-snip snip) → boolean?

  snip : (is-a?/c snip%)
Specification:

规格：要求释放指定的剪切。如果此管理员不是剪切的所有者，或者剪切无法释放，则返回#f。否则返回#t且剪切将不再拥有。

另请参见release-snip in editor<%>。

如果给定的剪切不由该管理员管理，则结果为#f。

Default implementation:

默认实现：返回#f。

method

(send a-snip-admin resized snip refresh?) → void?

  snip : (is-a?/c snip%)
  refresh? : any/c
Specification:

规范：由剪切调用，通知管理员指定的剪切已更改其显示大小。通常需要在调整大小后更新剪切，但剪切决定是否立即更新。

如果refresh?不是#f，则剪切请求立即更新，就像调用needs-update一样。否则，最终也必须调用needs-update。

如果给定的剪切不由该管理员管理，则忽略方法调用。

Default implementation:

默认实现：不执行任何操作。

method

(send a-snip-admin scroll-to snip    
  localx    
  localy    
  w    
  h    
  refresh?    
  [bias]) → boolean?
  snip : (is-a?/c snip%)
  localx : real?
  localy : real?
  w : (and/c real? (not/c negative?))
  h : (and/c real? (not/c negative?))
  refresh? : any/c
  bias : (or/c 'start 'end 'none) = 'none
Specification:

规范：由剪切调用以请求滚动，以便给定区域可见。剪切通常需要在滚动后更新，但剪切决定是否立即更新。

localx、localy、w和h参数指定要由滚动显示的剪切区域（在剪切坐标中）。

如果refresh?不是#f，则编辑器请求立即更新。

bias参数是：

  • 'start ——如果范围不适合可见区域，则显示左上角区域

  • 'none ——无特殊滚动说明

  • 'end ——如果范围不适合可见区域，则显示右下方区域

如果滚动编辑器，则结果为#t，否则为#f。

如果给定的剪切不由该管理员管理，则忽略方法调用（结果为#f）。

Default implementation:

默认实现：返回#f。

method

method

(send a-snip-admin get-tabs [length    
  tab-width    
  in-units]) → (listof real?)
  length : (or/c (box/c exact-nonnegative-integer?) #f) = #f
  tab-width : (or/c (box/c real?) #f) = #f
  in-units : (or/c (box/c any/c) #f) = #f
Specification: 规范：以列表形式返回当前选项卡位置数组。

The length box is filled with 选项卡数组的长度（因此返回的列表）, unless length is #f. The tab-width box is filled with 用于选项卡的宽度超过了选项卡数组的结尾, unless tab-width is #f. The in-units box is filled with 如果标签以画布单位指定，则为#t；如果标签以空格宽度指定，则为#f, unless in-units is #f.

Default implementation: 默认实现：返回空值。
