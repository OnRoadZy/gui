class

style-delta% : class?

  superclass: object%

style-delta%对象封装了样式更改。可以用delta表示的变化包括：

  • 更改字体系列

  • 更改字体表面

  • 将字体大小更改为新值

  • 将字体增大一个加法量

  • 将字体增大一个乘法量等

  • 更改字体样式（普通、斜体或倾斜）

  • 切换字体样式

  • 如果字体当前倾斜，则将其更改为斜体等

  • 更改字体粗细等

  • 更改下划线等

  • 更改垂直对齐方式等

  • 更改前景颜色

  • 使前景颜色变暗或变亮等

  • 更改背景色等

  • 更改文字背景透明度

set-delta方法对于大多数样式的delta设置都很方便；它采用高级delta规范并设置内部delta信息。

要充分利用样式增量，必须了解可通过诸如set-weight-on之类的方法操作的内部开/关设置。例如，字体粗细更改是通过内部设置的weight-on和weight-off来指定的。大致上，weight-on会在不存在权重设置时打开权重设置，weight-off会在存在权重设置时关闭权重设置。这两种方式的交互方式如下：

  • 如果weight-on和weight-off都设置为'base，则字体粗细不会更改。

  • 如果weight-on不是'base，则将粗体设置为weight-on。

  • 如果weight-off不是'base，则当基本样具有粗体'normal时，粗体将设置回'normal。

  • 如果weight-on和weight-off都设置为相同的值，则相对于该值切换粗体：如果基本样式具有weight-on，则粗体更改为'normal；如果基本样式具有不同的粗体，则更改为weight-on。

  • 如果同时设置了weight-on和weight-off，但值不同，则仅当基本样式weight-off时，才会将粗体更改为weight-on。

字体样式、平滑、下划线和对齐以类似的方式工作。

alignment-on和alignment-off的可能值为：

  • 'base

  • 'top

  • 'center

  • 'bottom

style-on和style-off的可能值为：

  • 'base

  • 'normal

  • 'italic

  • 'slant

smoothing-on和smoothing-off的可能值为：

  • 'base

  • 'default

  • 'partly-smoothed

  • 'smoothed

  • 'unsmoothed

underlined-on和underlined-off的可能值为：

  • #f (相当于'base)

  • #t

size-in-pixels-on和size-in-pixels-off的可能值为：

  • #f (相当于'base)

  • #t

transparent-text-backing-on和transparent-text-backing-off的可能值为：

  • #f (相当于'base)

  • #t

weight-on和weight-off 的可能值为：

  • 'base

  • 'normal

  • 'bold

  • 'light

样式增量中的系列和表面设置相互依赖：

  • 当一个delta的表面（face）是#f，它的系列（family）是'base，那么这表面和系列都不会被delta所改变。

  • 当一delta的表面是一个字符串并且它的系列是'base时，那么只有表面是由delta修改的。

  • 当delta的系列不是'base时，表面和系列都会被delta修改。如果delta的表面为#f，则应用delta将样式的面设置为#f，以便在选择字体时使用系列设置。

使用高级规范配置增量。返回值是delta本身。

除了'change-nothing和'change-normal之外，该命令只更改delta的一部分。因此，应用'change-bold和'change-italic设置样式和权重更改的增量。

change-command参数指定增量的更改方式；可能的值为：

  • 'change-nothing——重置所有更改

  • 'change-normal——关闭所有样式和大小调整

  • 'change-toggle-underline——为当前没有下划线的区域加下划线，反之亦然

  • 'change-toggle-size-in-pixels——为当前以点为单位的区域解释以像素为单位的大小，反之亦然

  • 'change-normal-color——分别将前景和背景更改为黑色和白色

  • 'change-italic——将字体样式更改为斜体

  • 'change-bold——将字体的粗体更改为粗体

  • 'change-family——更改字体系列（param是一个系列；请参阅font%）；另请参阅get-family

  • 'change-style——更改字体的样式（param是样式；请参阅font%）

  • 'change-toggle-style——切换字体样式（param是一种样式；请参见font%）

  • 'change-weight——更改字体的粗体（param是粗体；请参阅font%）

  • 'change-toggle-weight——切换字体的权重（param是权重；请参阅font%）

  • 'change-smoothing——更改字体的平滑（param是平滑；请参见font%）

  • 'change-toggle-smoothing）——切换字体的平滑（param是平滑；请参见font%）

  • 'change-alignment——更改对齐方式（param是对齐方式；请参见style-delta%

  • 'change-size——将大小更改为绝对值（param是大小）

  • 'change-bigger——使文本更大（param是一个加法量）

  • 'change-smaller——使文本变小（param是一个加法量）

  • 'change-underline——将下划线状态设置为带下划线或纯下划线

  • 'change-size-in-pixels——将大小解释设置为像素或点
