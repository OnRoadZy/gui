class

clipboard-client% : class?

  superclass: object%

clipboard-client%对象允许程序接管剪贴板，并为剪贴板数据的请求提供服务。有关详细信息，请参阅clipboard<%>.

clipboard-client%对象在成为当前客户端时与事件空间相关联；有关详细信息，请参阅set-clipboard-client。

constructor

(new clipboard-client%) → (is-a?/c clipboard-client%)

创建支持无数据格式的剪贴板客户端。

method

(send a-clipboard-client add-type format) → void?

  format : string?

将新的数据格式名称添加到剪贴板客户端支持的列表中。

format字符串通常是四个大写字母。(在Mac OS上，只有format的四个字符曾被使用。)例如，"TEXT"是UTF-8编码字符串格式的名称。新的格式名称可用于通信特定于应用程序和平台的数据格式。

method

(send a-clipboard-client get-data format)

 → (or/c bytes? string? #f)
  format : string?

当进程请求剪贴板数据而此客户端是剪贴板的当前客户端时调用。将请求的格式传递给方法，结果应为与请求的格式匹配的字节字符串，或者如果无法满足请求，则为#f。

只有客户端列表中的数据格式名称才会传递给此方法；请参见add-type。

当剪贴板调用此方法时，当前的事件空间与客户端的事件空间相同。如果在剪贴板请求点，当前的事件空间不是客户端的事件空间，那么当前线程就保证是客户端事件空间的处理程序线程。

method

(send a-clipboard-client get-types) → (listof string?)

返回剪贴板客户端支持的数据格式的名称列表。

method

(send a-clipboard-client on-replaced) → void?

当剪贴板客户机被视为剪贴板所有者时调用（因为剪贴板已被其他客户端或外部应用程序占用）。
