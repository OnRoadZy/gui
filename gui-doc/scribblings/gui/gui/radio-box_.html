class

radio-box% : class?

  superclass: object%

  extends: control<%>

radio-box%控件允许用户从多个互斥项中选择一个。这些项目显示为带标签的单选按钮的垂直列或水平行。与list-control<%>不同，不能动态更改radio-box%中的项目集。

每当用户更改选定的单选按钮时，都会调用单选框的回调过程。当创建每个单选框时，将提供一个回调过程作为初始化参数。

创建带有字符串或位图标签的单选按钮集。choices列表指定单选按钮标签；选项列表必须是同构的，可以是所有字符串，也可以是所有位图。

如果&出现在label中，则特别解析为button%。

choices中的每个字符串还可以包含一个&，它为单击相应的单选按钮创建一个助记键。对于label，将&&转换为&。

如果一个位图有一个掩码如果 choices是一个位图列表、和（请参见get-loaded-mask in bitmap%）与位图的大小相同，然后将遮罩用于标签。在用作标签时修改位图显示的标签上未指定的效果。

如果label是一个字符串，它将用作无线盒的标签。否则，单选框不会显示其标签。

当用户更改单选按钮选择时，调用callback过程（使用事件类型'radio-box）。

对于垂直排列的单选按钮集合，style参数必须包含'vertical，对于水平排列也要包含'horizontal。如果style包括'vertical-label，则单选框使用控件上方的标签创建；如果style不包括'vertical-label（可选包括'horizontal-label），然后标签创建在单选框的左侧。 如果style包括'deleted，则单选框将创建为隐藏的，并且它不会影响其父级的几何体；单选框稍后可以通过调用parent的add-child方法激活。。

默认情况下，最初选中第一个单选按钮。如果selection为正或#f，则传递到set-selection以设置初始单选按钮选择。

font参数确定控件的字体。 有关enabled参数的信息，请参见window<%>。 有关horiz-margin和vert-margin的信息参数，请参见subarea<%>。 有关min-width、min-height、stretchable-width、stretchable-height以及stretchable-height参数，请参见area<%>。

method

(send a-radio-box enable enable?) → void?

  enable? : any/c
(send a-radio-box enable n enable?) → void?
  n : exact-nonnegative-integer?
  enable? : any/c
Overrides enable in window<%>.

如果只提供一个参数，则启用或禁用整个单选框。

如果提供了两个参数，那么如果enable?为#f，则禁用第n个单选按钮，否则将启用该按钮（假定启用了整个单选框）。单选按钮从0开始编号。如果n等于或大于单选框中的单选按钮数，an exn:fail:contract exception is raised。

method

(send a-radio-box get-item-label n) → string?

  n : exact-nonnegative-integer?

按位置获取单选按钮的标签。单选按钮从0开始编号。如果n等于或大于单选框中的单选按钮数，an exn:fail:contract exception is raised。

method

(send a-radio-box get-item-plain-label n) → string?

  n : exact-nonnegative-integer?

与get-item-label类似，只是标签必须是字符串，并且移除标签中的 &。

method

(send a-radio-box get-number) → exact-nonnegative-integer?

返回单选框中的单选按钮数。

method

(send a-radio-box get-selection)

 → (or/c exact-nonnegative-integer? #f)

获取所选单选按钮的位置，如果未选择任何按钮，则返回#f。单选按钮从0开始编号。

method

(send a-radio-box is-enabled?) → boolean?

(send a-radio-box is-enabled? n) → boolean?
  n : exact-nonnegative-integer?
Overrides is-enabled? in window<%>.

如果没有提供任何参数，则报告整个无线电设备的启用状态。

否则，如果禁用第n个单选按钮（与禁用整个单选框无关），则返回#f，否则返回#t。单选按钮从0开始编号。如果n等于或大于单选框中的单选按钮数，an exn:fail:contract exception is raised。

method

(send a-radio-box set-selection n) → void?

  n : (or/c exact-nonnegative-integer? #f)

按位置设置选定的单选按钮，或者如果n为#f（不调用控件的回调过程），则取消选择所有单选按钮。单选按钮的编号从0开始。如果n等于或大于单选框中的单选按钮数，an exn:fail:contract exception is raised。

单选框选项可以被用户单击控件更改，这样的更改不会通过这个方法；使用control回调过程（作为初始化参数提供）来监视选项更改。
