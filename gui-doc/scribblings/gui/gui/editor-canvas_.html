class

editor-canvas% : class?

  superclass: object%

  extends: canvas<%>

一个editor-canvas%对象管理和显示一个text%或pasteboard%对象。

constructor

(new editor-canvas% 
    [parent parent] 
    [[editor editor] 
    [style style] 
    [scrolls-per-page scrolls-per-page] 
    [label label] 
    [wheel-step wheel-step] 
    [line-count line-count] 
    [horizontal-inset horizontal-inset] 
    [vertical-inset vertical-inset] 
    [enabled enabled] 
    [vert-margin vert-margin] 
    [horiz-margin horiz-margin] 
    [min-width min-width] 
    [min-height min-height] 
    [stretchable-width stretchable-width] 
    [stretchable-height stretchable-height]]) 
 → (is-a?/c editor-canvas%)
  parent : 
(or/c (is-a?/c frame%) (is-a?/c dialog%)
      (is-a?/c panel%) (is-a?/c pane%))
  editor : (or/c (or/c (is-a?/c text%) (is-a?/c pasteboard%)) #f)
   = #f
  style : 
(listof (or/c 'no-border 'control-border 'combo
              'no-hscroll 'no-vscroll
              'hide-hscroll 'hide-vscroll
              'auto-vscroll 'auto-hscroll
              'resize-corner 'no-focus 'deleted
              'transparent))
 = null
  scrolls-per-page : (integer-in 1 10000) = 100
  label : (or/c label-string? #f) = #f
  wheel-step : (or/c (integer-in 1 10000) #f) = 3
  line-count : (or/c (integer-in 1 1000) #f) = #f
  horizontal-inset : spacing-integer? = 5
  vertical-inset : spacing-integer? = 5
  enabled : any/c = #t
  vert-margin : spacing-integer? = 0
  horiz-margin : spacing-integer? = 0
  min-width : (or/c dimension-integer? #f) = #f
  min-height : (or/c dimension-integer? #f) = #f
  stretchable-width : any/c = #t
  stretchable-height : any/c = #t

如果一个画布用#f对editor初始化，安装一个编辑器晚于set-editor。

style列表可以容纳以下的标志：

  • 'no-border ——省略画布周围的边框；

  • 'control-border——为画布提供类似于text-field%控件的边框

  • 'combo——为画布提供一个类似于combo-field%控件的组合按钮;此样式用于'control-border、'hide-hscroll和'hide-hscroll

  • 'no-hscroll——不允许水平滚动并隐藏水平滚动条

  • 'no-vscroll——不允许垂直滚动并隐藏垂直滚动条

  • 'hide-hscroll——允许水平滚动,但隐藏水平滚动条

  • 'hide-vscroll——允许垂直滚动,但隐藏垂直滚动条

  • 'auto-hscroll——不需要时自动隐藏水平滚动条(除非指定'no-hscroll或'hide-hscroll

  • 'auto-vscroll——不需要时自动隐藏垂直滚动条(除非指定'no-hscroll或'hide-hscroll

  • 'resize-corner——在画布右下角只有一个滚动条可见时为调整控件留出空间

  • 'no-focus——阻止画布在单击画布或调用focus方法时接受键盘焦点

  • 'deleted——将画布创建为初始隐藏状态，而不影响parent的几何图形；稍后可以通过调用parent的add-child方法激活画布

  • 'transparent——在使用其父窗口的背景进行更新之前，画布被“擦除”；有关'transparent和屏幕外缓冲交互的信息，请参见canvas<%>

文本编辑器的垂直滚动基于行，而水平滚动和粘贴板垂直滚动基于每个水平页固定的步数。scrolls-per-page参数设置此值。

如果提供，则将wheel-step步进参数传递给wheel-step方法。通过'GRacket:wheelStep 偏爱可以全局覆盖默认的滚轮步进；请参见Preferences。

如果line-count不是#f，则将其传递给set-line-count方法。

如果horizontal-inset不是5，则传递给horizontal-inset方法。同样，如果vertical-inset不是5，则传递给vertical-inset方法。

有关enabled参数的信息，请参见window<%>。 有关horiz-margin和vert-margin的信息参数，请参见subarea<%>。 有关min-width、min-height、stretchable-width、stretchable-height以及stretchable-height参数，请参见area<%>。

method

(send an-editor-canvas allow-scroll-to-last) → boolean?

(send an-editor-canvas allow-scroll-to-last on?) → void?
  on? : any/c

启用或禁用最后一行滚动，或获取当前启用状态。如果启用了最后一行滚动，则可以滚动显示在此画布中的编辑器，使最后一行文本位于画布的顶部（或启用了基于底部的滚动时画布的底部；请参见scroll-with-bottom-base）。默认情况下，只能滚动编辑器，直到最后一行位于画布的底部（或顶部）。

method

(send an-editor-canvas allow-tab-exit) → boolean?

(send an-editor-canvas allow-tab-exit on?) → void?
  on? : any/c

获取或设置是否为编辑器画布启用选项卡退出。启用选项卡退出后，用户可以使用选项卡和箭头键将键盘焦点移出编辑器，使用Enter/Return键调用默认按钮，或使用Escape调用对话框的关闭操作。默认情况下，禁用选项卡退出。

当编辑器画布启用了选项卡退出时，Tab和Enter键盘事件将由框架的默认on-traverse-char方法使用；此外，对话框的默认方法使用转义键事件。否则，on-traverse-char允许键盘事件传播到画布。

method

(send an-editor-canvas call-as-primary-owner f) → any

  f : (-> any)

调用thunk并返回值。调用thunk时，如果画布有编辑器，则编辑器的get-admin方法将返回此画布的管理员。只有当编辑器显示在多个画布中时，此方法才有用。

method

(send an-editor-canvas force-display-focus) → boolean?

(send an-editor-canvas force-display-focus on?) → void?
  on? : any/c

启用或禁用强制聚焦模式。在强制聚焦模式下，即使画布没有键盘焦点，也会绘制显示在此画布中的插入符号或编辑器选择。

 → (or/c (or/c (is-a?/c text%) (is-a?/c pasteboard%)) #f)

返回此画布当前显示的编辑器，如果画布没有编辑器，则返回#f。

method

(send an-editor-canvas get-line-count)

 → (or/c (integer-in 1 1000) #f)

返回使用set-line-count安装的行计数，如果未设置最小行计数，则返回#f。

method

(send an-editor-canvas get-scroll-via-copy) → boolean?

如果滚动触发编辑器内容的副本（然后刷新新公开的内容），则返回#t。当滚动触发整个编辑器画布的刷新时返回#f。默认为#f。

另请参见on-scroll-to和after-scroll-to。

获取或设置画布中保留在编辑器内容左右两侧的像素数。默认值为5。

method

(send an-editor-canvas lazy-refresh) → boolean?

(send an-editor-canvas lazy-refresh on?) → void?

启用或禁用延迟刷新模式，或获取当前启用状态。在惰性刷新模式下，当需要更新窗口而不是在on-paint时调用画布的refresh方法。默认情况下，editor-canvas%对象不处于延迟刷新模式。

Overrides on-focus in window<%>.

method

(send an-editor-canvas on-paint) → void?

Overrides on-paint in canvas<%>.

重新绘制编辑器，如果没有显示编辑器，则清除画布。

在清除编辑器周围的边距后调用此方法，除非使用'transparent样式创建画布，但不会自动清除编辑器区域。换句话说，默认情况下，editor-canvas%更新与canvas%使用'no-autoclear样式更新类似，只是始终清除编辑器区域周围的空白。

method

(send an-editor-canvas scroll-to localx    
  localy    
  w    
  h    
  refresh?    
  [bias]) → boolean?
  localx : real?
  localy : real?
  w : (and/c real? (not/c negative?))
  h : (and/c real? (not/c negative?))
  refresh? : any/c
  bias : (or/c 'start 'end 'none) = 'none

请求滚动以使当前显示的编辑器中的给定区域可见。

localx、localy、w和h参数指定要由滚动显示的编辑器区域（在编辑器坐标中）。

如果refresh?不是#f，则在成功滚动后立即更新编辑器。

bias参数是以下之一：

  • 'start——如果范围不适合可见区域，则显示左上角区域

  • 'none——无特殊滚动说明

  • 'end——如果范围不适合可见区域，则显示右下方区域

如果滚动显示，返回值为#t；如果不滚动，返回值为#f（可能是因为请求的区域已经可见，因为显示的大小为零，也可能是因为编辑器当前正在打印）。

method

(send an-editor-canvas scroll-with-bottom-base) → boolean?

(send an-editor-canvas scroll-with-bottom-base on?) → void?
  on? : any/c

启用或禁用底基滚动，或获取当前启用状态。如果启用了基于底部的滚动，则滚动位置由与可视区域底部对齐的线边界（而不是与可视区域顶部对齐）确定。如果还启用了最后一行滚动（请参见allow-scroll-to-last），则即使编辑器没有填充可视区域，编辑器也会在显示区域中底端对齐。

method

(send an-editor-canvas set-editor edit    
  [redraw?]) → void?
  edit : (or/c (or/c (is-a?/c text%) (is-a?/c pasteboard%)) #f)
  redraw? : any/c = #t

设置画布显示的编辑器，释放当前编辑器（如果有）。如果新的编辑器已经有了一个与editor-canvas%无关的管理员，则新的编辑器不会安装到画布中。

如果重redraw?是#f，则不会立即绘制编辑器；在这种情况下，必须稍后强制重新绘制（例如，调用on-paint方法）。

如果画布上安装了带有set-line-count的行数，则会调整画布的最小高度。

method

(send an-editor-canvas set-line-count count) → void?

  count : (or/c (integer-in 1 1000) #f)

设置画布的图形最小高度以显示特定数量的文本行。行高是通过测量显示的编辑器第一行的顶部和底部之间的差异来确定的。在画布获得编辑器之前，不会更改最小高度。更改画布编辑器时，将重新计算最小高度。

如果行数设置为#f，则画布的图形最小高度将恢复为其原始值。

method

(send an-editor-canvas set-scroll-via-copy scroll-via-copy?)

 → void?
  scroll-via-copy? : any/c
更改滚动模式刷新。另请参见get-scroll-via-copy。

 → (or/c (integer-in 1 10000) #f)
