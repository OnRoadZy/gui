interface

selectable-menu-item<%> : interface?

  implements: labelled-menu-item<%>

selectable-menu-item<%>对象是用户可以选择的labelled-menu-item<%>。它还可以有一个键盘快捷键；快捷键显示在菜单中，当按下快捷键组合时，菜单框架中的默认on-subwindow-char方法将发送到菜单项。

method

(send a-selectable-menu-item command event) → void?

  event : (is-a?/c control-event%)

调用菜单项的回调过程，该过程在创建menu-item%或checkable-menu-item%的实例时提供。

method

(send a-selectable-menu-item get-shortcut)

 → (or/c char? symbol? #f)

获取菜单项的键盘快捷键字符或虚拟键。此字符或键与快捷方式前缀组合在一起，快捷方式前缀由get-shortcut-prefix报告。

如果菜单项没有快捷方式，则返回#f。

菜单项名称的快捷方式部分不包含在由get-label返回的标签中。

有关允许的键符号列表，请参见get-key-code in key-event%，但不允许使用以下项：'shift、'control、'numlock、'scroll、'wheel-up、'wheel-down、'release和'press。

method

(send a-selectable-menu-item get-shortcut-prefix)

 → 
(and/c (listof (or/c 'alt 'cmd 'meta 'ctl
                     'shift 'option))
       (λ (x) (implies (equal? 'unix (system-type))
                       (not (and (member 'alt x)
                                 (member 'meta x)))))
       (λ (x) (equal? x (remove-duplicates x))))

返回指示用于菜单项的键盘快捷方式的键盘前缀的符号列表。列表允许的符号如下：

  • 'alt——Meta（仅限Windows和X）

  • 'cmd——Command（仅限Mac OS）

  • 'meta——Meta（仅限Unix）

  • 'ctl——Control

  • 'shift——Shift

  • 'option——Option（仅限Mac OS）

在Unix上，最多只能提供'alt和'meta中的一个；'alt和'meta之间的唯一区别是组合键在菜单中的显示。

默认快捷方式前缀可从get-default-shortcut-prefix中获得。

快捷键由get-shortcut确定，使用通常报告的键代码或其他Shift/AltGr键代码（由get-other-shift-key-code in key-event%等生成）匹配键事件。当快捷键是键代码符号或ASCII字母或数字时，快捷键只匹配前缀中列出的修改键的确切组合。但是，对于除ASCII字母和数字以外的字符快捷方式，快捷方式前缀仅确定修改键的最小集合，因为访问字符可能需要其他修改器；例外情况是，在Windows或Unix，Alt/Meta键必须与前缀完全匹配（即是否包含）。在所有情况下，最精确的匹配优先；有关匹配排名的更多信息，请参阅map-function in keymap%。

空列表可用于快捷方式前缀。但是，只有当键事件包含非Shift修饰符或功能键时，默认的on-menu-char in frame%方法才会检查菜单快捷方式。因此，只有当快捷键是功能键时，空的快捷键前缀通常才有用。

method

(send a-selectable-menu-item set-shortcut shortcut) → void?

  shortcut : (or/c char? symbol? #f)

设置菜单项的键盘快捷字符。有关详细信息，请参阅get-shortcut。

如果快捷键字符设置为#f，则菜单项没有键盘快捷键。

method

(send a-selectable-menu-item set-shortcut-prefix prefix)

 → void?
  prefix : 
(and/c (listof (or/c 'alt 'cmd 'meta 'ctl
                     'shift 'option))
       (λ (x) (implies (equal? 'unix (system-type))
                       (not (and (member 'alt x)
                                 (member 'meta x)))))
       (λ (x) (equal? x (remove-duplicates x))))

设置符号列表以指示用于菜单项的键盘快捷方式的键盘前缀。

有关详细信息，请参见get-shortcut-prefix。
