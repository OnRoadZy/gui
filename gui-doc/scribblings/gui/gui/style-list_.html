class

style-list% : class?

  superclass: object%

style-list%对象包含一组style<%>对象，并维护它们之间的层次关系。style<%>对象只能通过style-list%对象的方法创建。有一个全局样式列表对象，即the-style-list，但是可以为单独的样式层次结构创建任意数量的独立列表。每个编辑器都创建自己的私有样式列表。

有关详细信息，请参见样式。

constructor

(new style-list%) → (is-a?/c style-list%)

将自动创建名为"Basic"的根样式。

method

(send a-style-list basic-style) → (is-a?/c style<%>)

返回根样式。每个样式列表都有自己的根样式。

有关'GRacket:default-font-size 偏爱的信息，请参见Preferences。

method

(send a-style-list convert style) → (is-a?/c style<%>)

  style : (is-a?/c style<%>)

将样式（可以是其他样式列表中的样式）转换为此列表中的style。如果style已在此列表中，则返回样式。如果style已命名，并且该名称的样式已在此列表中，则返回现有的命名样式。否则，将通过转换其基本样式（如果style是联接样式，则转换样式），然后在此列表中创建新样式来转换样式。

method

(send a-style-list find-named-style name)

 → (or/c (is-a?/c style<%>) #f)
  name : string?

按名称查找样式。如果找不到这样的样式，则返回#f。

method

(send a-style-list find-or-create-join-style base-style 
  shift-style) 
 → (is-a?/c style<%>)
  base-style : (is-a?/c style<%>)
  shift-style : (is-a?/c style<%>)

创建新的联接样式，或查找适当的现有联接样式。返回的样式始终未命名。有关详细信息，请参见样式。

base-style参数必须是此样式列表中的样式。

method

(send a-style-list find-or-create-style base-style 
  delta) 
 → (is-a?/c style<%>)
  base-style : (is-a?/c style<%>)
  delta : (is-a?/c style-delta%)

创建新的派生样式，或查找适当的现有样式。返回的样式始终未命名。有关详细信息，请参见样式。

base-style参数必须是此样式列表中的样式。如果base-style不是联接样式，如果它没有名称，并且它的delta可以用delta折叠（请参见collapse in style-delta%），则使用折叠的delta代替delta，使用基样式的base-style代替base-style；这种基样式的折叠和替换是递归执行的。

method

(send a-style-list forget-notification key) → void?

  key : any/c

请参见notify-on-change。

key参数是notify-on-change返回的值。

method

(send a-style-list index-to-style i)

 → (or/c (is-a?/c style<%>) #f)
  i : exact-nonnegative-integer?

返回与给定索引关联的样式，对于错误的索引返回#f。另请参见style-to-index。

method

(send a-style-list new-named-style name 
  like-style) 
 → (is-a?/c style<%>)
  name : string?
  like-style : (is-a?/c style<%>)

创建新的命名样式，除非该名称已被使用。

如果name已经被使用，那么like-style将被忽略，并返回与该名称关联的旧样式。否则，将为具有相同特征（即相同的基本样式和相同的样式增量或移位样式）的name创建一个新样式作为like-style。

like-style必须在此样式列表中，否则命名样式是从具有空样式增量的基样式派生的。

method

(send a-style-list notify-on-change f) → any/c

  f : ((or/c (is-a?/c style<%>) #f) . -> . any)

将回调f附加到样式列表。每次修改样式时都会调用回调f。

通常，一个样式中的更改会触发其他几个派生样式中的更改；为了允许客户端处理批处理中的所有更改，在处理一组样式后作为更改样式把#f传递给f。

notify-on-change的返回值是用于forget-notification的不透明键。

回调f替换它是equal?的任何回调,这有助于避免冗余注册时的冗余通知。回调f仅被弱保留（在make-weak-box的意义上），但只要f模拟的任何值是可访问的，它就被保留；例如，如果f表示应用了协定的函数，则只要原始（预定合约）函数是可访问的，就保留f以供通知。只要可以访问由notify-on-change生成的不透明密钥，回调f也会保留。

method

(send a-style-list number) → exact-nonnegative-integer?

返回列表中的样式数。

method

(send a-style-list replace-named-style name 
  like-style) 
 → (is-a?/c style<%>)
  name : string?
  like-style : (is-a?/c style<%>)

与new-named-style类似，但如果名称已映射到样式，则替换现有映射。

method

(send a-style-list style-to-index style)

 → (or/c exact-nonnegative-integer? #f)
  style : (is-a?/c style<%>)

返回特定样式的索引。样式的基样式（如果是连接样式，则为偏移样式）的索引保证低于样式自己的索引。（因此，根样式的索引始终为0。）每当向列表中添加新样式或更改其他样式的基样式或移位样式时，样式的索引都会发生更改。

如果给定的样式不在此列表中，则返回#f。
