class

editor-stream-in-base% : class?

  superclass: object%

editor-stream-in%对象使用editor-stream-in-base%对象执行低级数据读取。

从未直接实例化editor-stream-in-base%类，但可以实例化editor-stream-in-bytes-base%派生类。新的派生类必须重写本节中描述的所有方法。

method

(send an-editor-stream-in-base bad?) → boolean?

如果从流中读取时出错，则返回#t，否则返回#f。

method

(send an-editor-stream-in-base read data)

 → exact-nonnegative-integer?
  data : (and/c vector? (not immutable?))

read-bytes类似，但用Latin-1字符填充提供的向量，而不是填充字节字符串。此方法默认通过read-bytes实现。

method

(send an-editor-stream-in-base read-bytes bstr)

 → exact-nonnegative-integer?
  bstr : (and/c bytes? (not immutable?))

读取字节以填充提供的字节字符串。返回值是读取的字节数，如果流被清空，该值可能小于请求的字节数。如果流被清空，下次调用bad?必须返回#t。

method

(send an-editor-stream-in-base read-byte) → (or/c byte? #f)

读取一个字节并返回它，或者如果没有更多的字节可用，则返回#f。此方法的默认实现使用read-bytes。

method

(send an-editor-stream-in-base seek pos) → void?

  pos : exact-nonnegative-integer?

移动到流中指定的绝对位置。

method

(send an-editor-stream-in-base skip n) → void?

  n : exact-nonnegative-integer?

跳过流中接下来的n个字符。

method

(send an-editor-stream-in-base tell)

 → exact-nonnegative-integer?

返回当前流位置。
