13 初始化库

 (require racket/gui/init) package: gui-lib
racket/gui/init库是GRacket的默认启动库。它重新导出racket/init和racket/gui/base库，并将current-load设置为使用text-editor-load-handler。

 (require racket/gui/interactive) package: gui-lib
类似于racket/interactive，但用于GRacket。此库可以通过在中修改"config.rktd"文件中的'gui-interactive-file进行更改(find-config-dir)。此外，如果文件"gui-interactive.rkt"存在于(find-system-path 'addon-dir)，则运行该文件，而不是安装范围内的图形交互模块。
此库运行(find-graphical-system-path 'init-file)文件（如果存在），而不是其(find-system-path 'init-file)。与racket/interactive不同，此库不启动xrepl。

Added in version 1.27 of package gui-lib.
