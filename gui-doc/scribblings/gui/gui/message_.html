class

message% : class?

  superclass: object%

  extends: control<%>

消息控件是静态文本行或静态位图。文本或位图与消息的标签相对应（请参见set-label）。

constructor

(new message% 
    [label label] 
    [parent parent] 
    [[style style] 
    [font font] 
    [enabled enabled] 
    [vert-margin vert-margin] 
    [horiz-margin horiz-margin] 
    [min-width min-width] 
    [min-height min-height] 
    [stretchable-width stretchable-width] 
    [stretchable-height stretchable-height] 
    [auto-resize auto-resize]]) 
 → (is-a?/c message%)
  label : 
(or/c label-string? (is-a?/c bitmap%)
      (or/c 'app 'caution 'stop))
  parent : 
(or/c (is-a?/c frame%) (is-a?/c dialog%)
      (is-a?/c panel%) (is-a?/c pane%))
  style : (listof (or/c 'deleted)) = null
  font : (is-a?/c font%) = normal-control-font
  enabled : any/c = #t
  vert-margin : spacing-integer? = 2
  horiz-margin : spacing-integer? = 2
  min-width : (or/c dimension-integer? #f) = #f
  min-height : (or/c dimension-integer? #f) = #f
  stretchable-width : any/c = #f
  stretchable-height : any/c = #f
  auto-resize : any/c = #f

创建最初显示label的字符串或位图消息。如果这个位图有一个掩码如果 label是一个位图、和（请参见get-loaded-mask in bitmap%）与位图的大小相同，然后将遮罩用于标签。在用作标签时修改位图显示的标签上未指定的效果。 用于label的'app、'caution或'stop符号表示图标；'app是应用程序图标（Windows和Mac OS）或通用的“信息”图标（X）；'caution表示警告标志图标，'stop表示停止标志图标。

如果&出现在label中，则会对其进行特殊分析；在Windows和X下，字符在显示的控件中，&后加下划线表示键盘助记键。（在Mac OS下，助记符下划线不显示。）助记符对消息是无意义的（就on-traverse-char in top-level-window<%>来说是有关联的），但它支持与其他控件类型的一致性。程序员可以给助记符赋予意义（例如，通过重写 on-traverse-char）。

如果style包括'deleted，则消息将创建为隐藏的，并且它不会影响其父级的几何体；消息稍后可以通过调用parent的add-child方法激活。

font参数确定控件的字体。 有关enabled参数的信息，请参见window<%>。 有关horiz-margin和vert-margin的信息参数，请参见subarea<%>。 有关min-width、min-height、stretchable-width、stretchable-height以及stretchable-height参数，请参见area<%>。

如果auto-resize不是#f，则自动调整大小最初是不可见的（请参见auto-resize），并且message%对象的图形最小大小（graphical minimum size）是尽可能小的。

method

(send a-message auto-resize) → boolean?

(send a-message auto-resize on?) → void?
  on? : any/c

报告或设置当通过set-label更改标签时，是否自动设置message%的min-width和min-height。

method

(send a-message set-label label) → void?

  label : (or/c label-string? (is-a?/c bitmap%))
Overrides set-label in window<%>.

当label为字符串时，与set-label in window<%>相同。

否则，设置位图消息的位图标签。如果这个位图有一个掩码因为 label是一个位图、（请参见get-loaded-mask in bitmap%）与位图的大小相同，然后将遮罩用于标签。在用作标签时修改位图显示的标签上未指定的效果。 仅安装位图标签如果控件最初是用位图标签创建的。
