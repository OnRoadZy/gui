class

event% : class?

  superclass: object%

event%对象包含有关控件、键盘、鼠标或滚动事件的信息。另请参见control-event%、key-event%、mouse-event%和scroll-event%。

constructor

(new event% [[time-stamp time-stamp]]) → (is-a?/c event%)

  time-stamp : exact-integer? = 0

有关time-stamp的信息，请参见get-time-stamp。

method

(send an-event get-time-stamp) → exact-integer?

返回事件发生的时间（毫秒）。此时间与Racket的current-milliseconds过程报告的时间兼容。

method

(send an-event set-time-stamp time) → void?

  time : exact-integer?

设置事件发生的时间（毫秒）。另请参见Racket的current-milliseconds。

如果提供的值超出平台特定的时间值范围，an exn:fail:contract exception is raised。
