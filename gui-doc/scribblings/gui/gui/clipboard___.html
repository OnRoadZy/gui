interface

clipboard<%> : interface?

一个单一的clipboard<%>对象，即the-clipboard，用于管理系统范围内的剪切和粘贴剪贴板的内容。

在Unix上，第二个clipboard<%>对象，the-x-selection-clipboard管理系统范围内X11选择的内容。但是，如果'GRacket:selectionAsClipboard 偏爱首选项（请参见Preferences）设置为非零真值，则the-clipboard始终与the-x-selection-clipboard相同，并且不使用系统范围的X11剪贴板。

在Windows和Mac OS上，the-x-selection-clipboard始终与the-clipboard相同。

可以通过以下两种方式之一将数据输入剪贴板：设置当前剪贴板字符串或字节字符串，或安装clipboard-client%对象。安装客户端后，对剪贴板数据的请求将定向到客户端。

一般数据总是以字节字符串的形式从剪贴板中检索。检索剪贴板数据时，数据类型字符串指定数据字符串的格式。不同剪贴板格式的可用性由当前剪贴板所有者决定。

method

(send a-clipboard get-clipboard-bitmap time)

 → (or/c (is-a?/c bitmap%) #f)
  time : exact-integer?

获取当前剪贴板内容作为位图(Windows、Mac OS)，如果剪贴板不包含位图，则返回#f。

有关事件空间和当前剪贴板客户端的信息，请参阅get-clipboard-data。

有关time参数的论述，请参见Cut and Paste Time Stamps。如果time超出平台特定的时间范围，an exn:fail:contract exception is raised。

method

(send a-clipboard get-clipboard-data format 
  time) 
 → (or/c bytes? string? #f)
  format : string?
  time : exact-integer?

获取特定格式的当前剪贴板内容，如果剪贴板不包含请求格式的数据,则返回#f。

如果剪贴板客户端与非当前事件空间相关联，则通过客户端事件空间中的回调事件检索数据。如果一秒钟内没有结果可用，则放弃请求并返回#f。

有关format的信息，请参阅add-type in clipboard-client%。

有关time参数的论述，请参见Cut and Paste Time Stamps。如果time超出平台特定的时间范围，an exn:fail:contract exception is raised。

method

(send a-clipboard get-clipboard-string time) → string?

  time : exact-integer?

获取当前剪贴板内容作为简单文本，如果剪贴板不包含任何文本，则返回""。

有关事件空间和当前剪贴板客户端的信息，请参阅get-clipboard-data。

有关time参数的论述，请参见Cut and Paste Time Stamps。如果时间超出平台特定的时间范围，an exn:fail:contract exception is raised。

method

(send a-clipboard same-clipboard-client? owner) → boolean?

  owner : (is-a?/c clipboard-client%)

如果owner当前拥有剪贴板，则返回#t，否则返回#f。

method

(send a-clipboard set-clipboard-bitmap new-bitmap    
  time) → void?
  new-bitmap : (is-a?/c bitmap%)
  time : exact-integer?

将当前剪贴板内容更改为new-bitmap（Windows、Mac OS），并释放当前剪贴板客户端（如果有）。

有关time参数的论述，请参见Cut and Paste Time Stamps。如果time超出平台特定的时间范围，an exn:fail:contract exception is raised。

method

(send a-clipboard set-clipboard-client new-owner    
  time) → void?
  new-owner : (is-a?/c clipboard-client%)
  time : exact-integer?

更改拥有剪贴板的客户端：将客户端设置为new-owner，并将new-owner与当前事件空间关联（由current-eventspace确定）。当客户端不再是当前客户端时，将删除事件空间关联。

有关time参数的论述，请参见Cut and Paste Time Stamps。如果time超出平台特定的时间范围，an exn:fail:contract exception is raised。

method

  new-text : string?
  time : exact-integer?

将当前剪贴板内容更改为new-text，并释放当前剪贴板客户端（如果有）。

有关time参数的论述,请参见Cut and Paste Time Stamps。如果time超出平台特定的时间范围,an exn:fail:contract exception is raised。
