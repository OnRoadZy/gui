class

pasteboard% : class?

  superclass: object%

  extends: editor<%>

pasteboard%对象是用于显示任意定位（location）剪切的编辑器。

constructor

(new pasteboard%) → (is-a?/c pasteboard%)

只有将编辑器附加到editor-canvas%对象或其他显示（display）上，编辑器才会显示。

为新编辑器创建了一个新的keymap%对象。另请参见get-keymap和set-keymap。

将为新编辑器创建新的style-list%对象。另请参见get-style-list和set-style-list。

method

(send a-pasteboard add-selected snip) → void?

  snip : (is-a?/c snip%)
(send a-pasteboard add-selected x y w h) → void?
  x : real?
  y : real?
  w : (and/c real? (not/c negative?))
  h : (and/c real? (not/c negative?))

选择剪切而不取消选择其他剪切。当给定坐标时，此方法选择与给定矩形相交的所有剪切（在编辑器坐标中）。

粘贴板中的选择可以被系统响应其他方法调用更改，这样的更改不会通过这个方法；使用on-select来监视选择更改。

method

(send a-pasteboard change-style [style snip]) → void?

  style : (or/c (is-a?/c style-delta%) (is-a?/c style<%>) #f)
   = #f
  snip : (or/c (is-a?/c snip%) #f) = #f

将snip样式更改为特定样式或应用样式增量。如果snip为#f，则所有当前选定的剪切都将更改。如果style为#f，则根据default-style-name使用默认样式。

要将大量剪切从一种样式更改为另一种样式，请考虑提供一个style<%>实例，而不是style-delta%实例。否则，对于每个剪切，change-style必须将style-delta%实例转换为style<%>实例；此转换同时消耗时间和（临时）内存。

如果style被提供：编辑器的样式列表必须包含style，否则样式不会改变。另请参见convert in style-list%。

method

(send a-pasteboard find-next-selected-snip start)

 → (or/c (is-a?/c snip%) #f)
  start : (or/c (is-a?/c snip%) #f)

返回编辑器中选定的下一个剪切，start后开始搜索。（请参阅编辑器结构和术语以了解有关粘贴板中的剪切顺序的信息。）如果start是#f，则搜索从编辑器中的第一个剪切开始（并因此返回第一个选定的剪切，如果选择了任何剪切）。如果没有更多选定的剪切可用，或者如果start不在粘贴板中，则返回#f。

method

(send a-pasteboard find-snip x y [after])

 → (or/c (is-a?/c snip%) #f)
  x : real?
  y : real?
  after : (or/c (is-a?/c snip%) #f) = #f

找到与给定定位（location）相交的最前面的剪切（在给定剪切之后）。请参阅编辑器结构和术语以了解有关粘贴板中的剪切顺序的信息。

x和y参数在编辑器坐标中。如果没有提供after，则返回x和y处的最前面的剪切，否则返回after后的最前面的剪切。如果after的剪切不在粘贴板中，则返回#f。

只有当显示编辑器时，结果才有效（见编辑器结构和术语）。当get-admin返回系统管理员（不是#f）。

method

(send a-pasteboard on-default-event event) → void?

  event : (is-a?/c mouse-event%)
Overrides on-default-event in editor<%>.

选择、拖动和调整剪切大小：

  • 点击一个剪切选择剪切。按住Shift键单击可使用剪切扩展当前选择。

  • 点击剪切之间的空白处，拖动一个选择框；松开鼠标按钮后，所有接触框的剪切都会被选中。按住Shift键单击将使用新的剪切扩展当前选择。

  • 双击一个剪切调用on-double-click。

  • 单击选定的剪切，将选定的剪切拖到新定位（location）。

  • 单击所选对象的关联选项卡可调整对象的大小。
