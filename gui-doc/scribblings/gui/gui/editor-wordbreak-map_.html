class

editor-wordbreak-map% : class?

  superclass: object%

editor-wordbreak-map%对象与text%对象一起使用，以指定默认分词函数的分词条件。另请参见set-wordbreak-map、get-wordbreak-map、find-wordbreak和set-wordbreak-func。

一个全局对象the-editor-wordbreak-map将自动创建，并用作所有text%对象的默认映射。

分词对象实现从每个字符到符号列表的映射。以下符号是列表的法定元素：

  • 'caret

  • 'line

  • 'selection

  • 'user1

  • 'user2

字符值中的标志表示字符在使用相应的原因搜索分隔符时不会打断单词。例如，如果存在'caret，则字符是插入符号移动字的非中断字符。（每个非中断字符流都是一个单个单词。）

constructor

(new editor-wordbreak-map%) → (is-a?/c editor-wordbreak-map%)

所有ASCII字母数字字符都用'(caret line selection)初始化。除-之外的所有其他ASCII非空白字符都用'(line)初始化。所有ASCII空白字符和-都用null初始化。

method

(send an-editor-wordbreak-map get-map char)

 → (listof (or/c 'caret 'line 'selection 'user1 'user2))
  char : char?

获取char的映射值。有关详细信息，请参阅editor-wordbreak-map%。
