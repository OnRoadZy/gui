interface

menu-item<%> : interface?

menu-item<%>对象是menu%、popup-menu%或menu-bar%中的元素。影响父级的操作——例如重命名项、删除项或在项旁边添加检查——是通过menu-item<%>对象完成的。

菜单项可以是separator-menu-item%对象（仅是分隔符），也可以是labelled-menu-item<%>对象；后者更具体地说是menu-item%（普通菜单项）、checkable-menu-item%（可选中菜单项）或menu%（子菜单）的实例。

method

(send a-menu-item delete) → void?

从其父项中删除该项。如果菜单项已被删除，则delete无效。

另请参见restore。

method

(send a-menu-item get-parent)

 → (or/c (is-a?/c menu%) (is-a?/c popup-menu%) (is-a?/c menu-bar%))

返回包含该项的菜单、弹出菜单或菜单栏。菜单项的父级是在创建菜单项时指定的，不能更改。

method

(send a-menu-item is-deleted?) → boolean?

如果菜单项从其父级删除，则返回#t，否则返回#f。

method

(send a-menu-item restore) → void?

将已删除的项添加回其父项。无论项目的原始位置如何，该项目始终还原到父级的末尾。如果当前未删除该项目，则restore无效。
