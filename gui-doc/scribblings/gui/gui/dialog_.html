class

dialog% : class?

  superclass: object%

  extends: top-level-window<%>

对话框是一个模式化的顶级窗口：当显示对话框时，对对话框事件空间中的所有其他顶级窗口禁用键和鼠标按下/释放事件。

constructor

    [[parent parent] 
    [width width] 
    [height height] 
    [x x] 
    [y y] 
    [style style]

    [spacing spacing] 
    [alignment alignment]

    [min-height min-height]

  parent : (or/c (is-a?/c frame%) (is-a?/c dialog%) #f) = #f
  width : (or/c dimension-integer? #f) = #f
  height : (or/c dimension-integer? #f) = #f
  x : (or/c dimension-integer? #f) = #f

  spacing : spacing-integer? = 0

   = '(center top)
  min-width : (or/c dimension-integer? #f) = #f
  min-height : (or/c dimension-integer? #f) = #f
  stretchable-width : any/c = #t
  stretchable-height : any/c = #t

label字符串用作对话框标题栏中的标题。如果对话框的标签已更改（请参见set-label），则会更新标题栏。

parent参数可以是#f或现有框架。在Windows上，如果parent是现有框架，则新对话框始终位于其父级的顶部。在Windows和Unix上，当对对话框的父级进行图标化时，会对其进行图标化。

如果parent为#f，则新对话框的事件空间为当前事件空间，由current-eventspace确定。否则，parent的事件空间是新对话框的事件空间。

如果width或height参数不是#f，则它指定对话框的初始大小（以像素为单位），假定它大于最小大小，否则使用最小大小。在Windows和Mac OS（以及某些Unix窗口管理器）上，对话框的大小不可调整。

如果x或y参数不是#f，则指定对话框的初始位置。否则，如果在显示对话框之前未设置任何位置，则该位置居中（如果不是#f，则相对于parent，否则是屏幕）。

style标志可调整某些平台上对话框的外观：

  • 'no-caption——省略对话框的标题栏（Windows）

  • 'resize-border——在窗口（Windows）周围添加可调整大小的边框、调整窗口大小的功能（Mac OS）或右下角的增大框（较旧的Mac OS）

  • 'no-sheet——为对话框使用可移动窗口，即使提供了父窗口（Mac OS）

  • 'close-button——在对话框的标题栏中包含一个通常不会包含的关闭按钮（Mac OS）

即使未显示对话框，也可能会在创建对话框时排队等待一些通知事件。因此，在处理某些事件或关闭对话框的事件空间之前，无法回收新对话框的资源（例如内存）。

有关enabled参数的信息，请参见window<%>。 有关border、spacing和alignment的信息参数，请参见area-container<%>。 有关min-width、min-height、stretchable-width、stretchable-height以及stretchable-height参数，请参见area<%>。

method

(send a-dialog show show?) → void?

  show? : any/c
Overrides show in top-level-window<%>.

如果show?为真，将显示对话框，并且在关闭对话框之前，事件空间中的所有框架（和其他对话框）都将被禁用。如果show?为假，则隐藏对话框，并重新启用其他框架和对话框（除非仍显示其他预先存在的对话框）。

如果show?为真，该方法不会立即返回。相反，它以yield循环直到发现在调用yield之间隐藏了对话框。内部信号量与yield一起使用，以避免繁忙的等待，并确保在隐藏对话框后，show方法尽快返回。

method

(send a-dialog show-without-yield) → void?

像(send a-dialog show #t)，但立即返回而不是yield。
