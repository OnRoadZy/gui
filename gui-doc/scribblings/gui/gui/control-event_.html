class

control-event% : class?

  superclass: event%

control-event%对象包含有关控件事件的信息。control-event%的实例始终提供给控件或菜单项回调过程。

constructor

(new control-event% 
    [event-type event-type] 
    [[time-stamp time-stamp]]) 
 → (is-a?/c control-event%)
  event-type : 
(or/c 'button 'check-box 'choice
      'list-box 'list-box-dclick 'list-box-column
      'text-field 'text-field-enter
      'menu 'slider 'radio-box 'tab-panel
      'menu-popdown 'menu-popdown-none)
  time-stamp : exact-integer? = 0

event-type参数是以下参数之一：

  • 'button——用于button%点击

  • 'check-box——用于check-box%切换

  • 'choice——用于choice%项目选择

  • 'list-box——用于list-box%选择和取消选择

  • 'list-box-dclick——用于list-box%双击

  • 'list-box-column——用于list-box%列在column-control-event%实例中单击

  • 'text-field——用于text-field%更改

  • 'text-field-enter——用于单行text-field%输入事件

  • 'menu——用于selectable-menu-item<%>回调

  • 'slider——用于slider%更改

  • 'radio-box——用于radio-box%选择更改

  • 'tab-panel——对于tab-panel%选项卡更改

  • 'menu-popdown——用于popup-menu%回调（已选择项）

  • 'menu-popdown-none——用于popup-menu%回调（未选择任何项）

此值是使用get-event-type方法从control-event%对象中提取的。

有关time-stamp的信息,请参见get-time-stamp。

method

(send a-control-event get-event-type)

(or/c 'button 'check-box 'choice
      'list-box 'list-box-dclick 'text-field
      'text-field-enter 'menu 'slider 'radio-box
      'menu-popdown 'menu-popdown-none 'tab-panel)
返回控件事件的类型。有关每个事件类型符号的信息，请参阅control-event%。

      'menu-popdown 'menu-popdown-none 'tab-panel)

设置事件类型。有关每个事件类型符号的信息，请参阅control-event%。
